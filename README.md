
## INSTALL

```
npm install
```
## USAGE

包含gulp自动化的常用插件：
    1. scss编译为css
    2. css压缩
    3. js压缩、公共js合并base.js
    4. 图片压缩
    5. html模块化<include引入>
    6. 静态资源版本号控制
    7. css sprites 雪碧图
    8. 字体（字体图标）copy
    9. 本地服务器并热更新
    10. babel
    11. css自动转换rem
    12.vue基本引入方式做移动端单页面
