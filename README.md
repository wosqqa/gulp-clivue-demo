
## INSTALL

```
npm install
```
##功能

```
包含一饮而尽自动化的常用插件：
    1. scss编译为css
    2. css压缩
    3. js压缩
    4. 图片压缩
    5. html模块化
    6. 静态资源版本号控制
    7. css sprites 雪碧图
    8. 字体（字体图标）copy
    9. 本地服务器并热更新
    10. babel
    11.直接使用750设计稿对应 px、打包后会自动转换为rem.

```
